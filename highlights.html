Requirement Analysis:

In this phase, Biswajit will understand the requirements from the business and prepare a requirement specification document. He establishes and maintains an agreement with the business users on what the system will do. He will also need to specify the functional and non-functional requirements of the system.

High Level Design:

In this phase, he will participate in defining the architectural changes of the system using UML diagrams which mentions the different system components and its interactions with other entities. Based on approved design, sequence diagram, and class diagram the construction or development phase of the project is initiated.

Construction phase of the project:

In this phase Biswajit along with offshore will start implementation of the design which involves activities like Web layer development using Spring MVC, the middleware layer development using Oracle SOA Suite and Oracle Service Bus and finally the backend implementation which involves WSDL development and web services development using Apache CXF frameworks. He keeps track of project milestones and report project status to his client Counterpart and Cognizant supervisor.

Project coordination with offshore:

He also coordinates the work of Cognizant offshore team which performs development activities. He performs code reviews and design reviews for the modules completed and help in resolving issues and providing updates to senior manages in the client side.

Deployment and Testing

In this phase he maintains and updates the deployment configurations for the modules to be sent to different stages or servers for deployment and does the smoke testing to ensure the deployment did not break any of the existing functionalities.
